第⼀步 安装 Hugo 解压⽂件 hugo_extended_0.88.1_Windows-64bit 将解压得到的三个⽂件放置到⾃⼰指定的安装⽬录 通过命令⾏进⼊安装⽬录，执⾏ hugo version 观察是否有版本信息输出，确认安装 结果 修改系统变量，将 hugo 的安装⽬录添加到系统变量中 第⼆步 创建博客本地⽂件夹 通过命令⾏前往你希望放置博客⽂件的⽬录 执⾏ hugo new site <blog_name> 命令，在当前⽬录下⽣成⼀个名字为 <blof_name> 的⽂件夹，这个⽂件夹将作为存放所有博客⽂件的⽬录 第三步 创建博客⽂章 通过命令⾏前往上⼀步中创建的博客⽬录 执⾏ hugo new posts/<post_name> 命令，创建⼀篇名字为 <post_name> 的博客 ⽂章 第四步 设置博客主题 解压⽂件 gohugo-theme-ananke-master 将解压得到的⽂件夹更名为 anake 复制重命名后的⽂件夹到博客本地⽂件夹的 themes ⽬录下 修改 config.toml ⽂件，添加⼀⾏ theme = “<theme_name>” ，其中 <theme_name> 填⼊主题的名字 第五步 本地预览博客 在命令⾏中前往创建的博客⽬录 执⾏ hugo server -D 命令 打开浏览器进⼊对应的⽹站进⾏预览（⽹站会在执⾏完命令后提示你） 第六步 安装 Git
