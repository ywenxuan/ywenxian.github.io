### **确保两个虚拟网卡的IP地址处于同一网段。**

​      **验证方法：**

**在物理主机ping虚拟机的Ip地址，同样在虚拟机中ping物理主机的IP地址**

**若无法ping通，物理机的虚拟网卡的Ip分配方式改为自动。**

|                      | 物理主机     | VM虚拟机                    |
| :------------------: | ------------ | --------------------------- |
| **命令行查看IP地址** | **ipconfig** | **ip addr show / ifconfig** |
|                      |              |                             |
